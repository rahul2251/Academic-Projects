CHAPTER 1: INTRODUCTION

1.1 Abstract

GreenCycle E-Waste Solutions is a web-based application designed to streamline the collection and recycling of electronic waste. In the current digital age, e-waste is accumulating at an alarming rate. This project provides a platform where users can schedule doorstep pickups for their old electronics, collectors can manage their pickup routes efficiently, and administrators can oversee the entire process with analytics. A key feature of this system is the integration of an AI-based E-Waste Scanner using TensorFlow.js, which allows users to identify waste items via image recognition instantly.

1.2 Existing System and Need for System

Existing System: Currently, e-waste disposal is largely unorganized. Citizens often sell scrap to informal vendors (kabadiwalas) who may not recycle items safely, leading to environmental hazards. There is no centralized tracking of how much waste is generated or recycled.
Need for System:

To provide a certified, eco-friendly disposal channel.

To offer convenience through doorstep pickup scheduling.

To track the lifecycle of e-waste from generation to recycling.

To ensure data security via certified destruction of storage devices.

1.3 Scope of System

The system covers three main modules:

User Module: Registration, Profile Management, AI Image Scanning, Pickup Scheduling, Feedback.

Collector Module: Task Assignment (Auto/Manual), Route Management, Status Updates (In-Transit/Completed).

Admin Module: Dashboard Analytics, User/Collector Management, Reporting (CSV Export), Request Approval.

1.5 Brief Description of Technology Used

PHP: Used for server-side logic, session management, and database interaction.

MySQL: Relational database management system used to store user data, requests, and feedback.

Tailwind CSS: A utility-first CSS framework used for designing a responsive and modern UI.

JavaScript: Used for client-side validation and dynamic DOM manipulation.

CHAPTER 2: PROPOSED SYSTEM

2.1 Feasibility Study

Technical Feasibility: The project uses standard web technologies (PHP/MySQL) which are open source and widely supported. The AI component runs client-side, reducing server load.

Operational Feasibility: The UI is designed to be simple and intuitive, requiring minimal training for users and collectors.

Economic Feasibility: Being a web-based solution, deployment costs are low. It eliminates the need for expensive physical infrastructure for initial collection booking.

2.2 Objectives of the proposed system

To automate the scheduling of e-waste pickups.

To provide real-time tracking of request status (Pending -> Accepted -> Completed).

To categorize e-waste accurately using AI.

To generate reports for environmental impact analysis.

2.3 Users of the system

Administrator: Superuser with full control over the system.

Collector: Field agent responsible for physical collection and status updates.

User (Household/Corporate): The entity generating e-waste and raising requests.

CHAPTER 5: LIMITATIONS OF SYSTEM
Internet Dependency: The system requires an active internet connection to load the AI models and submit requests.

Lighting Conditions: The AI scanner's accuracy depends on the quality and lighting of the uploaded image.

Physical Verification: The system relies on the user's input for item weight/quantity; actual verification happens only upon collection.

CHAPTER 6: PROPOSED ENHANCEMENTS
Real-time GPS Tracking: Integrate Google Maps API to allow users to track the collector's vehicle in real-time.

Reward Points System: Implement a gamification module where users earn "Green Points" for recycling, redeemable for coupons.

Mobile Application: Develop a dedicated Android/iOS app for offline support and push notifications.

Chatbot Integration: Add an AI chatbot to answer FAQs about recyclable materials.

CHAPTER 7: CONCLUSION
The GreenCycle E-Waste Solutions project successfully addresses the problem of unorganized e-waste disposal. By leveraging modern web technologies and AI, we have created a user-friendly platform that connects households with authorized recyclers. The system brings transparency, efficiency, and accountability to the waste management process, contributing significantly to a cleaner and greener environment.
